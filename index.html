Grace Hopper

Sobre

Grace Hopper (1906-1992) foi uma pioneira da ciência da computação e uma das primeiras programadoras de computadores. Ela é conhecida por seu trabalho na linguagem de programação COBOL e por suas contribuições para o desenvolvimento do primeiro compilador, um software que traduz código fonte para código de máquina.

Contribuições

Grace Hopper fez várias contribuições significativas para o campo da ciência da computação, incluindo:

  • O desenvolvimento do primeiro compilador, o A-0 System, que foi o precursor das linguagens de programação modernas.
  • A criação da primeira linguagem de programação de alto nível, o COBOL, que se tornou uma das linguagens de programação mais amplamente utilizadas na indústria.
  • O desenvolvimento do conceito de "depuração" de software, que se tornou um termo comum para se referir à correção de erros em programas de computador.

Cronologia

1906

 Nascimento de Grace Hopper em Nova York, Estados Unidos.

1944

 Ingresso na Marinha dos Estados Unidos como parte do programa WAVES (Women Accepted for Volunteer Emergency Service).

1949

 Conclusão do programa de doutorado em matemática na Universidade Yale, tornando-se uma das primeiras mulheres a obter um doutorado em matemática nos Estados Unidos.

1952

 Desenvolvimento do primeiro compilador, o A-0 System.

1959

 Desenvolvimento do COBOL, a primeira linguagem de programação de alto nível.

1986

 Reforma do posto de almirante na Marinha dos Estados Unidos, tornando-se a primeira mulher a alcançar esse posto.

1992

 Falecimento de Grace Hopper aos 85 anos.
